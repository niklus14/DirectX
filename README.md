# AZCON ONE

Static hackathon prototype for the AZCON ONE smart mobility assistant.

## Run

```bash
node server.js
```

Then open:

```text
http://localhost:5173/
```

## Included Flows

- Tourist arriving in Azerbaijan.
- Local user flying abroad.
- ADY Baku-Ganja round trip.
- eSIM tariff cards with provider redirect links.
- AZCON Pass concept mock.
- OpenRouteService route integration through the local server.
- Groq intent extraction through the local server.
